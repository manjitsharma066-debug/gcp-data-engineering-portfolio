# 🚀 End-to-End Batch ETL Pipeline using Google Cloud Platform (GCP)

## 📌 Project Overview

This project demonstrates an end-to-end Batch ETL Pipeline built using Google Cloud Platform (GCP). The pipeline extracts sales data from a CSV file, performs data transformation using Python, stores the processed data in Google Cloud Storage, and loads it into BigQuery for analytics and reporting.

---

## 🎯 Business Problem

Organizations receive sales data daily from multiple sources. Manual processing is slow and error-prone.

The objective of this project is to automate the ETL process and build a centralized analytics platform using GCP services.

## ⚙️ ETL Workflow

1. Read raw CSV file
2. Clean and transform data using Python
3. Upload processed data to Cloud Storage
4. Create External Table
5. Create Native Table
6. Execute Business SQL Queries
7. Generate Analytics
