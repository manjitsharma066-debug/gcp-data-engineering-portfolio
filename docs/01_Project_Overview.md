# Project Overview

## Project Name

End-to-End Batch ETL Pipeline using Google Cloud Platform (GCP)

---

## Objective

The objective of this project is to build an end-to-end batch ETL pipeline that extracts sales data from a CSV file, transforms it using Python, stores it in Google Cloud Storage, and loads it into BigQuery for reporting and analytics.

---

## Business Scenario

A retail company receives daily sales data in CSV format.

The business requires an automated solution to:

- Clean incoming sales data
- Store raw and processed files securely
- Load data into BigQuery
- Generate business reports using SQL
