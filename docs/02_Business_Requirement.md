# Business Requirement

## Problem Statement

The organization receives daily sales data from different business locations.

Manual processing causes delays and increases the chances of errors.

The company wants an automated ETL pipeline on Google Cloud Platform.

---

## Functional Requirements

- Read sales CSV
- Validate records
- Clean data
- Upload to Cloud Storage
- Create External Table
- Create Native Table
- Execute Business SQL Queries

---

## Expected Output

- Centralized analytics platform
- Faster reporting
- Better data quality
- Scalable cloud solution
